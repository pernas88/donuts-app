# Donut’s App
**Autor:** Gabriel Pernas González
  Aplicación fullstack para gestión de Donut’s con CRUD completo, favoritos,
  Redux Toolkit y una UI cuidada con Tailwind.
  ## Requisitos
  - Node 18+
  - MongoDB local o Atlas
## Arranque
```bash
# 1) Backend
  cd server
  cp .env.example .env
  npm i
  npm run dev
# 2) Frontend
cd ../client
npm i
  npm run dev
